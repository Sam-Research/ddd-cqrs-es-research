# DDD/CQRS/ES 架构的调研

> DDD Domain Driven Design

> CQRS表示Command Query Responsibility Segregation，即命令和查询责任分离，是由Greg Young提出的一种将系统的读（查询）、写（命令）操作分离为两种独立子系统的架构模式。在一些应用场景下，这种分离是很有价值的，但要注意，CQRS对大多数系统而言，让系统变得更加复杂。
